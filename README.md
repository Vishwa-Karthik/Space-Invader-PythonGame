# Space-Invader Python Game

## A simple python game developed from [PyGame](https://www.pygame.org/)

## Installation
```bash
pip install pygame
```

## Usage
### Initialise the init module from pygame
```python
pygame.init()
```

### Set the screen size
```python
size = 800,400
screen = pygame.display.set_mode(size)
```

### Load the images
```python
pygame.image.load('filename.format')
```
